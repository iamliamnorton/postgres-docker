# Postgres Docker

[![Circle CI](https://circleci.com/gh/iamliamnorton/postgres-docker.png?style=shield)](https://circleci.com/gh/iamliamnorton/postgres-docker)

This is a Postgres 9.4 image.

## Building

    $ make build

## Shell

Access to bash.

    $ make shell

## Test

Test image.

    $ make test
